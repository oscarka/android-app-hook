# 2️⃣ Hook 引擎（核心能力）

## 👉 技术选型

* 主：Frida
* 辅：Xposed Framework

👉 推荐：先用 Frida（开发效率高）

---

## ✅ 2.1 启动方式

```bash
frida -U -f com.taobao -l hook.js
```

或嵌入 frida-server

---

## ✅ 2.2 Hook 网络层（最重要）

### 示例（OkHttp）

```javascript
Java.perform(function () {
    var Request = Java.use('okhttp3.Request');

    Request.Builder.build.implementation = function () {
        var request = this.build();

        console.log("URL:", request.url().toString());

        return request;
    };
});
```

👉 你能拿到：

* URL
* 参数
* headers

---

## ✅ 2.3 Hook 参数（关键）

```javascript
var RealCall = Java.use('okhttp3.RealCall');

RealCall.execute.implementation = function () {
    var response = this.execute();
    console.log("Response:", response.toString());
    return response;
};
```

---

## ✅ 2.4 Hook 业务函数（更高级）

```javascript
Java.perform(function () {
    var Cart = Java.use("com.taobao.cart.CartManager");

    Cart.addItem.implementation = function (itemId) {
        console.log("Add to cart:", itemId);
        return this.addItem(itemId);
    };
});
```

# 4️⃣ Agent Runtime（执行调度）

你要写一个执行器：

```python
def execute(action):
    if api_exists(action):
        call_api()
    elif intent_exists():
        call_intent()
    else:
        fallback_ui()
```

---

# 5️⃣ OpenClaw 接入（关键）

你要定义：

```json
{
  "functions": [
    {
      "name": "search_product",
      "description": "搜索商品",
      "parameters": {
        "keyword": "string"
      }
    }
  ]
}
```

---

👉 OpenClaw 输出：

```json
{
  "function": "search_product",
  "arguments": {
    "keyword": "羽绒服"
  }
}
```

---
